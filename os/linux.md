# Linux Operating System

## History

- Linus Torvalds

## Characteristics

- Open-source based on UNIX (Unix-like)
- Free
- Portable
- Multi-user
- Multitasking
- Support CLIs
  - bash
  - zsh
  - tcsh
- Scalable
- Highly customizable
- Cluster computing architecture
- Monolithic kernel
- File systems
  - xfs
  - ramfs
  - vfat
  - cramfsm
  - ext3
  - ext4
  - ext2
  - ext1
  - autofs
  - devpts
  - ntfs

## Popular distributions

- Red Hat Enterprise Linux (RHEL)
- CentOS
- Ubuntu
- Linux Mint

## Disadvantages

- Not ease-to-use for new users
- Lack of applications
- Lack of hardware-supported
