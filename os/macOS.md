# Macintosh Operating System - macOS

*Old names: Mac OS X and OS X*

Mach kernel and BSD -> NeXTSTEP -> Mac OS X

## File systems

- APFS (Apple File System)
- Read-only NTFS (New Technology File System)

## Components

- Drawin (core)
  - Mach kernal
  - FreeBSD

## Frameworks

- Cocoa
- Carbon
